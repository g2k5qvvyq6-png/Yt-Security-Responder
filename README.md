##  项目简介

本项目是自动化安全事件分析与智能响应 Agent仓库。

我们专注于网络安全产品开发，为解决企业安全运营中的 **告警疲劳** 与 **响应延迟** 痛点，利用大模型技术构建了一款自动化安全事件分析与响应 Agent。该 Agent 具备**长链推理**与**多 Agent 协作**能力，能够自动研判告警、重构攻击链并执行阻断动作。

> 本项目展示内容与申请表单中的描述完全一致，所有逻辑均已内部验证。

##  解决的核心痛点

- **告警疲劳**：每天数千条告警，90% 为误报或低优先级，分析师被淹没。
- **响应延迟**：从发现真实攻击到处置平均需要 40 分钟，攻击者横向移动仅需几分钟。
- **专家依赖**：高水平安全专家稀缺，无法 7×24 覆盖。

##  核心特性

| 模块 | 功能描述 |
|------|----------|
| **研判 Agent** | 基于威胁情报与资产信息，对原始告警进行归一化、富化，并输出置信度评分（0-100）。 |
| **溯源 Agent** | 自动重构攻击链（MITRE ATT&CK），向前追溯 30 分钟行为，向后探查失陷主机。 |
| **决策执行 Agent** | 调用防火墙 API 封禁 IP、下发 EDR 隔离主机，并通过飞书/Jira 通告分析师。 |
| **人机环回** | 关键操作需人工确认，常规封堵 2 秒内自动完成。 |
